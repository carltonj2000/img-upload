# Simple Image Upload with React

The code in this repository is based on the
[Simple Image Upload with React](https://codeburst.io/react-image-upload-with-kittens-cc96430eaece)
article and the
[File Upload With Multer in Node.js and Express](https://code.tutsplus.com/tutorials/file-upload-with-multer-in-node--cms-32088)
article.

To run the code.

- `yarn start` in the server directory
- `yarn start` in the root directory for debugging (locallhost:3000)
- `yarn build` in the rood directory for production (localhost:5000)
